# Radix Converter

A React web application to convert numbers between different bases: decimal, hexadecimal, binary, and octal.

## Features

- Convert between multiple number bases:
  - **Decimal** (Base 10)
  - **Hexadecimal** (Base 16)
  - **Binary** (Base 2)
  - **Octal** (Base 8)
- Input validation with clear error messages
- Press **Enter** to convert
- Clear button to reset all fields

## Getting Started

### Prerequisites

Install [Node.js](https://nodejs.org/) (v18 or later).

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for production

```bash
npm run build
```
